## Open source

Lidraughts is a rewrite of [lichess/lila](https://github.com/ornicar/lila/) for draughts. The entire platform — server, UI, and draughts engine bindings — is open source under the AGPL-3.0 license.

## Supported variants

| Variant | Board | Description |
|---|---|---|
| Standard | 10×10 | International draughts — the classic game |
| Frisian | 10×10 | Kings and pieces can capture in all directions |
| Antidraughts | 10×10 | The goal is to lose all your pieces |
| Frysk! | 10×10 | Starts with only 5 pieces per side |
| Breakthrough | 10×10 | First player to reach the opponent's back rank wins |
| Russian | 8×8 | Russian draughts on an 8×8 board |
| Brazilian | 8×8 | Similar to Russian with FMJD rules |
